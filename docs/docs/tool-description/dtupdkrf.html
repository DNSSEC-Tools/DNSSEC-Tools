NAME

dtupdkrf - Ensures that a keyrec file is in the current format

SYNOPSIS

  dtupdkrf [options] <keyrec files>

DESCRIPTION

dtupdkrf ensures that a keyrec file is in the current format. If the keyrec file has an old format, it is converted to the current format.

The format of keyrec files does not change very often. Therefore, dtupdkrf should be run when installing new versions of the DNSSEC-Tools software. It does not have to be run very often otherwise.

The number of error fixed in the given keyrec files will be given as dtupdkrf's exit code.

WARNING: dtupdkrf should NOT be run on keyrec files that are in active use by other DNSSEC-Tools programs. You MUST stop rollerd, zonesigner, or any other DNSSEC-Tools programs BEFORE running dtupdkrf on active files.

OPTIONS

dtupdkrf supports several types of options. These options are detailed below.

-count

Display the count of keyrecs in each keyrec file.

-quiet

Display no information. -quiet and -verbose are mutually exclusive.

-verbose

Display verbose information. -verbose and -quiet are mutually exclusive.

-Version

Display dtupdkrf' version information and exit.

-help

Display a usage message and exit.

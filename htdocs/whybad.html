You are insecure

So you told me, but why?

Because the bumper image is loaded from a domain name which is insecure. It has dnssec data for the A record for the web server, but the A record actually contains a bad signature. A browser that was properly protected, either within the browser itself or because your site's recursive validating resolver was secured, would prevent you from retrieving this image since the zone date for the web server was known to be bad. For a list of our test records, see our Test Zone page.

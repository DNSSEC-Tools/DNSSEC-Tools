
Several Perl modules have been developed for DNSSEC-Tools to assist in
maintaining DNSSEC-secured domains.  These routines manipulate DNSSEC-Tools
files, provide GUI interfaces, and manipulate common command options.

These DNSSEC-Tools interfaces are:

\begin{description}

\item{\perlmod{QWPrimitives.pm.tex}} - \perlmod{QWizard} primitives

\item{\perlmod{conf.pm.tex}} - DNSSEC-Tools configuration file routines

\item{\perlmod{keyrec.pm.tex}} - Routines to manipulate \path{keyrec} files

\item{\perlmod{timetrans.pm.tex}} - Interfaces to convert an integer seconds
count into text units

\item{\perlmod{tooloptions.pm.tex}} - DNSSEC-Tools common option routines

\end{description}

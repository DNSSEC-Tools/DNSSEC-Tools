\subsection{\bf QWPrimitives.pm Module}

{\bf NAME}

\perlmod{Net::DNS::SEC::Tools::QWPrimitives.pm} - \perlmod{QWizard} primitives
for DNSSEC-Tools

{\bf SYNOPSIS}

\begin{verbatim}
    use Net::DNS::SEC::Tools::QWPrimitives;
    use Getopt::Long::GUI;

    GetOptions(  ...,
                 ['GUI:nootherargs',1],
                 ['GUI:otherprimaries',dnssec_tools_get_qwprimitives()],
                 ['GUI:submodules','getzonefiles','getzonenames'],
              );
\end{verbatim}

{\bf DESCRIPTION}

\perlmod{QWizard} is a dynamic GUI-construction kit.  It displays a series of
questions, then retrieves and acts upon the answers.  This module provides
access to \perlmod{QWizard} for DNSSEC-Tools software.

In particular, the \func{dnssec\_tools\_get\_qwprimitives()} returns a set of
primary screens for requesting a set of zone files followed by a set of domain
names for those zone files.  These are then pushed into the \var{\_\_otherargs}
\var{qwparam} variable, which is used by \perlmod{Getopt::GUI::Long} to
generate the \var{\@ARGV} list.
